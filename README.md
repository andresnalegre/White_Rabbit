## About

**White Rabbit** is an immersive, Matrix-themed interactive experience built entirely with Vanilla JS and CSS. It opens with a cinematic intro video, followed by a typewriter sequence, and transitions into a full Matrix rain canvas with a glitch text effect — all driven by layered audio and smooth animations.

No frameworks, no dependencies. Just pure browser APIs.

## Features

- Click-to-start black overlay with animated prompt
- Cinematic intro video with fade transition
- Typewriter effect with pooled audio playback
- Interactive message sequence responding to click, touch and keyboard
- Full-screen Matrix rain canvas with katakana and symbol characters
- Glitch text animation using Canvas API
- Layered audio: typewriter sounds and Matrix background music
- Developer info card with social links
- Fully responsive across mobile, tablet and desktop

## Run locally

### Requirements

- Any modern browser
- A local server (e.g. VS Code Live Server or `npx serve`)

### Setup

```bash
git clone https://github.com/andresnalegre/White_Rabbit
cd White_Rabbit
npx serve .
```

Then open `http://localhost:3000` in your browser.
